This project is a **console-based application developed in C++** for the *Algorithms* course.  
The system provides comprehensive management of Olympic-style sports competitions, including secure access, competition simulation, and advanced statistics with historical medal tables.

## ğŸš€ Main Features

- **Custom Security System**  
  Access is controlled using a student ID (legajo) and a dynamically generated password based on a specific numeric algorithm.

- **Persistent Data Management**  
  Uses binary files to store and retrieve the `archivoCompetencia` structure, which contains information about **87 different sports**.

- **Competition Simulation**  
  Randomly generates the number of participants (between 4 and 16 countries) for each discipline, ensuring no duplicate countries within the same competition.

- **Advanced Statistics Module**
  - Medal tables by country and by sport
  - Overall ranking using a weighted medal points system
  - Podium search using filters (e.g., initial letter of the sport)
  - Console pagination for improved data visualization and user experience
